# Курсовая работа по дисциплине «Базы данных»

## Тема
Разработка базы данных для автоматизации учета в клубе любителей собаководства

## Описание
Информационная система предназначена для хранения и обработки данных о собаках, породах, выставках, болезнях, родословных и вязках.  
База данных реализована с использованием PostgreSQL и содержит механизмы обеспечения целостности данных, триггеры, функции и индексы.

## Используемые технологии
- PostgreSQL
- SQL
- Qt
- QtSql
- C++

## Структура базы данных

В базе данных реализованы следующие таблицы:

1. `породы`
2. `собаки`
3. `вязки`
4. `родители`
5. `выставки`
6. `болезни`
7. `история болезней`

## Реализованные возможности

- создание и удаление таблиц;
- связи между таблицами через FOREIGN KEY;
- ограничения PRIMARY KEY, UNIQUE, CHECK, DEFAULT;
- триггеры PostgreSQL;
- пользовательские функции;
- индексы для ускорения запросов;
- заполнение тестовыми данными;
- аналитические SQL-запросы.

## Особенности базы данных

### Триггеры
Реализованы триггеры:
- проверки пола родителей;
- проверки пола участников вязки.

### Пользовательские функции
Реализована функция:
- проверки родства собак перед вязкой.

### Индексы
Созданы индексы:
- `idx_exhib`
- `idx_disease`
- `idx_parents`
